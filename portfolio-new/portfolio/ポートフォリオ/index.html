自己紹介

私についてのご紹介と、大切にしている想いです。

はじめまして、北海道・小樽を拠点に活動している Yasu と申します。
HTML / CSS を軸に、無駄を省いた「ミニマルで見やすいWebサイト」の制作を得意としています。

これまでにクローンサイト制作に取り組む中で、
デザインの再現性から細部のあしらいまで、確かなスキルを磨いてまいりました。

私が最も大切にしているモットーは、最新のAIをフル活用した「丁寧に、速く」という姿勢です。
お客様の想いやご要望をしっかり汲み取るため、誠実で温かいコミュニケーションを何よりも重視しています。

ただ見栄えが良いだけでなく、長く愛される高品質なWebサイトを心を込めてお届けいたします。
どんな小さなご相談でも、どうぞお気軽にお声がけください。

よくある質問

お客様からよくいただくご質問です。

Q

納期はどれくらいですか？

A

制作内容によって異なりますが、LPであれば7〜10日程度が目安です。
ヒアリング後に、より正確なスケジュールをご案内いたします。

Q

修正は可能ですか？

A

3回まで無料で対応しています。
「イメージと違った」「色味を調整したい」など、遠慮なくお申し付けください。

Q

デザインからお願いできますか？

A

はい、可能です。
参考サイトやイメージを共有いただければ、ミニマルで見やすいデザインをご提案いたします。
